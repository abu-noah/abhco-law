## TECHNICAL IMPROVEMENTS V1 → V2

| Area | V1 | V2 |
|---|---|---|
| CSS architecture | Flat | CSS custom properties (brand tokens), logical properties |
| Responsive | Basic | Mobile-first, fluid `clamp()` values, 3 breakpoints |

| SEO meta | Minimal | Full Open Graph, bilingual meta description, keywords |
